# Portfolio Optimization

This project explores several numerical optimization methods applied to portfolio
construction, from basic least-squares problems to Markowitz mean-variance
optimization and higher-moment (kurtosis-aware) portfolio selection. It compares
solvers (CVXPY, SciPy's SLSQP) and custom implementations (SQP, SCP) in terms of
correctness, speed, and scalability.

It is derived from an academic optimization project carried out at **ENSTA Paris**.

## Contents

- `Supporting_functions.py` — shared utilities: downloading market data (via
  `yfinance`), computing return statistics, Monte Carlo portfolio sampling,
  plotting helpers, and the core portfolio-optimization routines (Markowitz,
  kurtosis-based SLSQP and SCP solvers).
- `cvxpy-least-squares.py` — solves a bounded least-squares problem with CVXPY
  and benchmarks solve time as problem size grows.
- `slsqp-least-squares.py` — same least-squares problem solved with SciPy's
  SLSQP, for comparison against the CVXPY approach.
- `markowitz-portfolio.py` — classic Markowitz mean-variance portfolio,
  evaluated for varying numbers of assets.
- `markowitz-portfolio-probabilistic.py` — a probabilistic (chance-constrained)
  variant of the Markowitz portfolio.
- `asset-time-series-plot.py` — plots cumulative returns of the assets over the
  training and testing periods.
- `montecarlo-portfolio-visualization.py` — visualizes random portfolios via
  Monte Carlo sampling alongside the computed optimal portfolios.
- `sqp_solver.py` — a from-scratch Sequential Quadratic Programming (SQP) solver
  applied to a non-convex constrained problem.
- `sqp-vs-slsqp.py` — compares the custom SQP solver against SciPy's SLSQP on
  the same non-convex problem (speed, iterations, constraint satisfaction).
- `kurtosis-scp-convergence.py` — solves the kurtosis-minimization problem with
  Sequential Convex Programming (SCP) and studies convergence from different
  starting points.
- `kurtosis-methods-comparison.py` — compares SLSQP, SCP, and Markowitz on the
  kurtosis-aware portfolio problem.
- `kurtosis-sdp.py` — formulates and solves the kurtosis problem as a
  semidefinite program (SDP) with CVXPY.
- `figures/` — plots and figures generated by the scripts above.
- `Markowitz_portfolio_optimization_report.pdf` — the written report
  summarizing methodology and results.

## Requirements

Python 3, with `numpy`, `pandas`, `matplotlib`, `scipy`, `cvxpy`, and `yfinance`
(used to download historical stock price data from Yahoo Finance).
